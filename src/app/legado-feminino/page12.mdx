# Legado Feminino

A API sobre 100 mulheres que marcaram a história é uma API que fornece informações sobre os feitos das principais mulheres históricas do mundo. A API responde com um código de status 200 para solicitações bem-sucedidas, fornecendo recursos de informações no formato JSON, cada um contendo informações detalhadas como ID referente a mulher, nome, ano morte, ano de nascimento e feito histórico.{{ className: 'lead' }}

Para solicitações inválidas, a API retorna um código de status 422, garantindo um feedback claro sobre erros nos parâmetros fornecidos.{{ className: 'lead' }}


## Autenticação

A API sobre 100 mulheres que marcaram a história não requer autenticação para acessar os dados.

## Rate Limit

A API sobre 100 mulheres que marcaram a história possui um limite de taxa de 100 solicitações por minuto. Se você exceder esse limite, receberá um código de status 429. O uso indiscriminado também poderá levar ao bloqueio do seu IP.

## Base URL

A URL base para a API sobre 100 mulheres que marcaram a história é `https://apis.codante.io/api/legadoFeminino`.

## O model Bloquinho

O model `Woman` contém todas as informações sobre um bloco de carnaval.

## Créditos e Fontes

Os dados fornecidos por esta API são baseados em informações coletadas do site [Blocos de Rua](https://www.blocosderua.com/), a maior plataforma de conteúdo sobre carnaval de rua do Brasil. O site original oferece informações detalhadas sobre blocos de carnaval em diversas cidades brasileiras, incluindo São Paulo, Rio de Janeiro, Belo Horizonte, Salvador, Florianópolis, Recife e Olinda, Brasília, Porto Alegre e Fortaleza.

Agradecemos à equipe do Blocos de Rua por disponibilizar essas informações que enriquecem a experiência dos foliões durante o Carnaval 2025.

Para acessar a fonte original dos dados e obter informações adicionais sobre os blocos de carnaval, visite [blocosderua.com](https://www.blocosderua.com/).
